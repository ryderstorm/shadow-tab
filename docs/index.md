Transform your New Tab into a sleek, customizable dark workspace.

Shadow Tab replaces the default New Tab page with a dark, minimal loading screen and then redirects you to a URL you choose (optionally after a delay).

## Features

- **Custom New Tab destination**: Set any URL to open on new tabs.
- **Optional redirect delay**: Add a short delay with a lightweight loading animation.
- **Dark theme by default**: Avoid the bright white flash.
- **Background color controls**: Choose a color (picker + presets) for both New Tab and Options.
